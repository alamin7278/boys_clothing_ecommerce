## Instructions

### Creating Diagrams

Use tools like:
- **draw.io** (https://app.diagrams.net/) - Free, web-based
- **Lucidchart** - Professional diagramming
- **Microsoft Visio** - Enterprise tool

### Creating Wireframes

Use tools like:
- **Figma** (https://www.figma.com/) - Free, collaborative
- **Balsamiq** - Sketch-style wireframes
- **draw.io** - Also supports wireframes

### File Naming Convention

- Use lowercase with underscores
- Be descriptive: `architecture_diagram.png`
- Include version if needed: `erd_diagram_v2.png`

### Export Settings

- **Format:** PNG or SVG (PNG preferred for documentation)
- **Resolution:** 300 DPI for print, 150 DPI for screen
- **Size:** Optimize file size while maintaining quality
