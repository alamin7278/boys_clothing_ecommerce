# Trello Board Setup Guide - Design Phase

## System Design & Architecture Phase Board Setup

### Board Name
**Second-Hand Boys Clothing E-Commerce - SE Design Phase**

### Lists Structure

1. **Backlog**
   - All design tasks identified
   - Requirements from SRS to be addressed

2. **To Do**
   - Design tasks ready to be worked on
   - Prioritized items from backlog

3. **In Progress**
   - Tasks currently being designed
   - Maximum 2-3 items per team member

4. **Review**
   - Completed designs awaiting team review
   - Documentation review items

5. **Done**
   - Completed and validated designs
   - Finalized documentation

## Tips

1. **Break Down Large Tasks:** Split complex tasks into smaller, manageable cards
2. **Use Checklists:** Add checklists within cards for multi-step tasks
3. **Attach Files:** Attach design files (diagrams, wireframes) to relevant cards
4. **Set Due Dates:** Assign due dates to track progress
5. **Regular Updates:** Update board status daily or at least weekly
6. **Team Communication:** Use card comments for team discussions
